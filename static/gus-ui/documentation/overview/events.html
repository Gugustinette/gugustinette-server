Overview - Events

Events

Events are the most important part of web components when it comes to programming an interface.
An event is handled using the "addEventListener()" function, which is used as the following :

  • 1. Select your element in JS
  • 2. Call the "addEventListener()" function on it
  • 3. Catch the specific event you want
  • 4. Write the code to read when the event is fired


For example, handling the state change of a gus-switch can be done like that (in JavaScript) :
var gusSwitch = document.querySelector("#gus-switch-selector")

gusSwitch.addEventListener("value-changed", function() {
    Write your code here
})
