CSS Parts

Parts are what make components easy to customize.
Selecting a part is fairly easy :

  • 1. Select the element as usual in CSS, for example with a class ".element"
  • 2. Add the "::part()" selector to you're element selector
  • 3. Precise the part you want to customize in the parenthesis.


For example, selecting all gus-switch "background" part can be done like that (in CSS) :
gus-switch::part(switch-background) {
    put your rules here
}
