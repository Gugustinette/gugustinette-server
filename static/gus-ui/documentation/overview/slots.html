Overview - Slots

Slots

Slots add a lot to the way you can customize and use the components. Slots are element to add inside a component in order to replace certain parts of it.

Some components take infinite amount of elements inside, such as the Section Slider or the Drop Down Section which mean you can just throw every element you wish inside.

Other components take specific slots, such as the Alert Box which takes 2 speficic slots that are "icon" and "close". Targeting a slot is fairly easy, all you need to do is putting your element inside the component and add the "slot" attribute to it.
