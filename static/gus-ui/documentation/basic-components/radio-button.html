Basic Component - Radio Button

Overview

Radio Button is a component that is used to make a selection option among a set of data. It is generally used with a radio button list.

Element

Radio Button is created in HTML as below :
code

Attributes

Radio Button has 2 attributes :
content (String): correspond to the text written beside the radio button
checked (Boolean): tells if the radio button is selected or not

CSS Parts

Radio Button has 2 parts to customize :
radiobutton: correspond to the radio button itself
radiobutton-content: correspond to the text written beside the radio button

Slots

Radio Button doesn't have any slot.

Events

Radio Button doesn't have any event to handle.
