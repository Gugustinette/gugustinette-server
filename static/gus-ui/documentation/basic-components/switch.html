Basic Component - Switch

Overview

Switch is a state component, that tells if something is on or off.

Element

Switch is created in HTML as below :
code

Attributes

Switch has 2 attributes :
checked (Boolean): tells if the switch is checked or not
disabled (Boolean): tells if the switch should be in "disabled" state or not

CSS Parts

Switch has 2 parts to customize :
switch-background: correspond to the background part of the switch
switch: correspond to the button of the switch

Slots

Switch doesn't have any slot.

Events

Switch has 1 event to handle :
value-changed: fired when the state of the switch changed
