Basic Component - Slider

Overview

Slider is a component that is used to select a value among a set of data.

Element

Slider is created in HTML as below :
code

Attributes

Slider has 6 attributes :
type (String): int | default -> correspond to the type of slider
elements (JSON): JSON array of strings representing every options
hide-scrollbar (Boolean): tells if the scrollbar on the side of the slider should be hided or not
int-start (Int): correspond to the first int of the range of number to display on a "Int" slider
int-end (Int): correspond to the last int of the range of number to display on a "Int" slider
selected-value (String | Int): correspond to the value selected in the slider

CSS Parts

Slider has 1 part to customize :
slider: correspond to the slider itself

Slots

Slider doesn't have any slot.

Events

Slider has 1 event to handle :
selected-value-changed: fired when the user selected a new value using the Slider
