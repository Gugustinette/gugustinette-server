Basic Component - Radio Button List

Overview

The Radio Button List is a placement component that is mean to contain one or many radio button.

Element

Radio Button List is created in HTML as below :
code

Attributes

Radio Button List has 2 attributes :
value (String): correspond to the value selected by the user in the radio button list
default-value (String): correspond to the default value that is selected when the radio buttons are rendered

CSS Parts

Radio Button List has 1 part to customize :
radio-button-list : correspond to the list iself

Slots

Radio Button List can contain as many radio buttons as you want

Events

Radio Button List has 1 event to handle :
value-changed : fired when the user changed the selected value in the list
