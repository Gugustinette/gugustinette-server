Basic Component - Progress Bar

Overview

Progress bar is a component that is mean to display the state of a process to the user.

code

Attributes

Progress bar has 4 attribute :
filling (Int): correspond to the value filled in the bar
max-filling (Int): correspond to the maximum value that can be filled in the bar (100 by default)
display (Boolean): tells if the percentage should be displayed on the bar
to-display (String): correspond to the text displaying on the bar, if you want to display your own text

CSS Parts

Progress bar has 3 parts to customize :
progress-bar: correspond to the progress bar itself
progress-bar-filler: correspond to the part that is filling the bar
progress-bar-display: correspond to the text displaying on the bar

Slots

Progress bar doesn't have any slot.

Events

Progress bar doesn't have any event to handle.
