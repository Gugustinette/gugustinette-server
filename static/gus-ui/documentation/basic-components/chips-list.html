Basic Component - Chips List

Overview

Chipslist is a placement component that is mean to contain one or many chips.

Element

Chipslist is created in HTML as below :
code

Attributes

Chipslist has 1 attribute, which correspond to the type of chips that is mean to go inside the list :
type (String): normal | check-chips | cross-chips

CSS Parts

Chipslist has 1 part to customize :
chips-list : correspond to the Chipslist itself

Slots

Chipslist can contain as many chips as you want

Events

Chipslist has 1 event to handle :
value-changed : fired when the state of the chips inside of the list changed
