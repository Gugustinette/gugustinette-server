Basic Component - Text Field

Overview

Text Field is an input component, that can only handle a single line of text.

Element

Text Field is created in HTML as below :
code

Attributes

Text Field has 1 attribute :
placeholder (String): correspond to the text displaying in the text field when empty

CSS Parts

Text Field has 1 part to customize :
text-field-input: correspond to the text field itself

Slots

Text Field has 1 slot :
icon: correspond to an icon that will be displayed at the left of the text field

Events

Text Field has 1 event to handle :
text-modif: fired when the text contained in the text field changed
