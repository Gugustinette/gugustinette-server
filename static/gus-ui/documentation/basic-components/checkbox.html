Basic Component - Checkbox

Overview

Checkbox is a state component which can be checked or not.

Element

Checkbox is created in HTML as below :
code

Attributes

Checkbox has 2 attributes :
checked (Boolean): tells if the checkbox is checked or not
content (String): standard text to display beside the checkbox

CSS Parts

Checkbox has 2 part to customize :
checkbox : correspond to the checkbox itself
checkbox-content : correspond to the text displaying below the checkbox (if any)

Slots

Checkbox doesn't have any slot.

Events

Button doesn't have any specific event to handle.
