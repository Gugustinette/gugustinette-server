Basic Component - Switch Box

Overview

Switch Box is a state component, that tells if something is on or off.

Element

Switch Box is created in HTML as below :
code

Attributes

Switch Box has 3 attributes :
content (String): correspond to the text written it the switch box
checked (Boolean): tells if the switch in the box is checked or not
disabled (Boolean): tells if the Switch Box should be in "disabled" state or not

CSS Parts

Switch Box has 4 parts to customize :
switch-box: correspond to the switch box itself
content: correspond to the text written in the switch box
switch-background: correspond to the background part of the switch
switch: correspond to the button of the switch

Slots

Switch Box doesn't have any slot.

Events

Switch Box has 1 event to handle :
value-changed: fired when the state of the switch box changed
