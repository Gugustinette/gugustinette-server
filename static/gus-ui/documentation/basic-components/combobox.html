Basic Component - Combobox

Overview

Combobox is a component that is mean to select one option among a set of data.

Attributes

Chips has 3 attributes :
content (JSON): should be a JSON Array containing every option as strings
default-value (String): represent the default option selected in the combobox (should be the first element in the array by default)
selected-value (String): represent the value selected by the user at any time.

CSS Parts

Combobox has 7 part to customize :
combobox: correspond to the combobox itself
combobox-content: correspond to the top part of the combobox, displaying the selected value
combobox-content-text: correspond to the text displaying the selected value
combobox-content-arrow: correspond to the arrow svg of the combobox
combobox-elements: correspond to the bottom part of the combobox, displaying every option when clicking
combobox-element: correspond to every element displaying in the bottom part
combobox-element-text: correspond to the text displaying in every element of the bottom part

Slots

Combobox doesn't have any slot.

Events

Combobox has 3 event to handle :
value-changed: fired when the state of the combobox changed, which mean the user made a selection
toggle: fired when the user toggle the combobox
close: fired when the user close the combobox
