Basic Component - Range Slider

Overview

Range Slider is a component that is used to select a value in a specific range of number.

Element

Range Slider is created in HTML as below :
code

Attributes

Range Slider has 4 attributes :
int-start (Int): correspond to the minimum value that can be selected
int-end (Int): correspond to the maximum value that can be selected
value (Int): correspond to the value selected by the user on the slider
round-button-position (Boolean): tells if the range slider should round the position of the button correspond to the selected value

CSS Parts

Range Slider has 2 parts to customize :
range-slider: correspond to the background part of the slider
button: correspond to the button of the slider

Slots

Range Slider doesn't have any slot.

Events

Range Slider has 1 event to handle :
selected-value-changed: fired when the user selected a new value using the range slider
