Basic Component - Chips

Overview

Chips is a component that is used to make selections, filter content, and others. It is generally used with a chips list.

Element

Chips is created in HTML as below :
code

Attributes

Chips has 5 attributes :
content (String): correspond to the text written in the chips
check-visible (Boolean): tells if the chips is of type "check", which mean it can be checked or not
checked (Boolean): tells if the chips is checked
cross-visible (Boolean): tells if the chips is of type "cross", which mean it has a cross to click on
auto-delete (Boolean): tells if the chips must be removed when clicking on the cross

CSS Parts

Chips has 4 part to customize :
chips: correspond to the chips itself
chips-check: correspond to the "check" svg displaying on "check-chips"
chips-content: correspond to the text written in the chips
chips-cross: correspond to the "cross" svg displaying on "cross-chips"

Slots

Chips doesn't have any slot.

Events

Chips has 2 event to handle :
value-changed : fired when the state of the chips changed (only work on "check-chips")
close : fired when the user clicked on the cross (only work on "cross-chips")
