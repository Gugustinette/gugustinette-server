Basic Component - Navbar

Overview

Navbar is a placement component that is used to navigate through the website or app easily.

Element

Navbar is created in HTML as below :
code

Attributes

Navbar has 1 attribute :
nb-element (int): represent the number of button inside the navbar

CSS Parts

Navbar has 1 part to customize :
navbar: correspond to the navbar itself

Slots

Navbar has 1 slot :
navbar-mobile-button: correspond to the button showing at the top of the screen on mobile

Events

Navbar doesn't have any event to handle.
