Advanced Component - Time Picker

Overview

Time Picker is a powerful component that let the user select a specific time.

Element

Time Picker is created in HTML as below :
code

Attributes

Time Picker has 2 attributes :
time (String): return the time selected by the user
period (String): am | pm -> correspond to the time period selected by the user

CSS Parts

Time Picker doesn't have any part to customize at the moment.

Slots

Time Picker doesn't have any slots.

Events

Time Picker has 6 event to handle :
confirm: fired when the user confirmed the selection of a time
cancel: fired when the user canceled the selection of a time
confirm-keyboard: fired when the user selected a time using the keyboard input
cancel-keyboard: fired when the user canceled the selection of a time using the keyboard input
confirm-keyboard-scfl: fired when the user selected a time succesfully using the keyboard input
confirm-keyboard-wrong: fired when the user failed in selecting a time using the keyboard input
