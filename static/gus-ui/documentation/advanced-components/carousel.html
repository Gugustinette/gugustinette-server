Advanced Component - Carousel

Overview

Carousel is a surface that display a set of images.

Element

code

Attributes

Carousel has 6 attributes :
image-collection (JSON): JSON Array of the URLs leading to the images to display in the carousel
infinite (Boolean): tells if the navigation through the images should be infinite or not
auto-scrolling (Boolean): tells if the carousel should navigate through the images automatically
auto-scroling-delay (Int): the time passing between each images when auto-scrolling is true (in ms)
hide-arrows (Boolean): tells if the arrows on the sides of the carousel must be hided or not
hide-bar (Boolean): tells if the state bar displaying at the bottom of the carousel must be hided or not

CSS Parts

Carousel has 4 parts to customize :
carousel: correspond to the carousel itself
left-button: correspond to the left arrow of the carousel
bar: correspond to the state bar at the bottom of the carousel
right-button: correspond to the right arrow of the carousel

Slots

Carousel has 3 slots :
left-button: correspond to the left button of the carousel
bar: correspond to the state bar at the bottom of the carousel
right-button: correspond to the right button of the carousel

Events

Carousel has 2 event to handle :
next-image: fired when the carousel is going to the next image of the collection succesfully
previous-image: fired when the carousel is going to the previous image of the collection succesfully
