Advanced Component - Card

Overview

Card is a surface that display content and actions on a single topic.

Element

Card is created in HTML as below :
code

Attributes

Card has 4 attributes :
label-title (String): correspond to the title of the card
label-description (String): correspond to the description of the card
label-text (String): correspond to a bonus text with low emphasis
media (Image): correspond to an image that should be displayed on the card

CSS Parts

Card has 7 parts to customize :
card: correspond to the Card itself
media: correspond to the media displaying on the card if any
content: correspond to the area that contain the title and description of the card
content-title: correspond to the title of the card
content-description: correspond to the description of the card
text: correspond to the low emphasied text of the card if any
buttons: correspond to the area displaying buttons on the card if any

Slots

Card can contain as many buttons as you wish on its bottom part.

Events

Card doesn't have any event to handle.
