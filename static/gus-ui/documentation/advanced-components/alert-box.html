Advanced Component - Alert Box

Overview

Alert Box is a information component that is used to alert the user of something.

Element

Alert Box is created in HTML as below :
code

Attributes

Alert Box has 2 attributes :
label-title (String): correspond to the title of the alert
label-description (String): correspond to the description of the alert

CSS Parts

Alert Box has 6 parts to customize :
alert-box: correspond to the alert box itself
icon: correspond to the icon of the alert box
title: correspond to the title of the alert box
line: correspond to the line showing between the title and description
description: correspond to the description of the alert box
close-icon: correspond to the "cross" icon displaying at the top-right corner of the alert box

Slots

Alert Box has 2 slots :
icon: correspond to the icon of the alert box
close: correspond to the "closing" icon displaying at the top-right corner of the alert box

Events

Alert Box doesn't have any event to handle.
