Advanced Component - Drop Down Section

Overview

Drop Down Section is a placement component that is used to show the elements inside only when clicking on it.

Element

Drop Down Section is created in HTML as below :
code

Attributes

Drop Down Section has 1 attribute :
content (String): correspond to the text written in the section

CSS Parts

Drop Down Section has 5 parts to customize :
drop-down-section: correspond to the whole section itself
drop-down-section-title: correspond to the area displaying permanently with the title and arrow
drop-down-section-title-text: correspond to the title of the section
drop-down-section-title-arrow: correspond to the arrow of the section
drop-down-section-content: correspond to the area holding the content of the section

Slots

Drop Down Section as many elements as you wish inside it.

Events

Drop Down Section has 1 event to handle.
statue-changed: fired when the statut of the section changed
