Advanced Component - Tooltip

Overview

Tooltip is a UI component that is used to show a small tip below a specific element.

Element

A tooltip should contain the element which is mean to have a tooltip. Tooltip is created in HTML as below :
code

Attributes

Tooltip has 3 attributes :
content (String): correspond to the text written in the tooltip
delay (Int): correspond to the delay time that pass before the tooltip is displayed (in ms)
follow-cursor (Boolean): tells if the tooltip should be displayed below the user's cursor

CSS Parts

Tooltip has 1 part to customize :
tooltip: correspond to the tooltip itself

Slots

Tooltip doesn't have any slot.

Events

Tooltip doesn't have any event to handle.
