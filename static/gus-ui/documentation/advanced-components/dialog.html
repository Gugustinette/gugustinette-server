Advanced Component - Dialog

Overview

Dialog is a information component that is used to inform or ask something ot the user.

Element

Dialog is created in HTML as below :
code

Attributes

Dialog has 2 attributes :
label-title (String): correspond to the title of the dialog
label-description (String): correspond to the description of the dialog

CSS Parts

Dialog has 5 parts to customize :
dialog: correspond to the dialog itself
content: correspond to the area holding the title and description of the dialog
content-title: correspond to the title of the dialog
content-description: correspond to the description of the Dialog
buttons: correspond to the area holding the buttons of the dialog

Slots

Dialog can contain as many buttons as you wish.

Events

Dialog doesn't have any event to handle.
