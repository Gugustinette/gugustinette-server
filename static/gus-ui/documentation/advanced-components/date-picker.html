Advanced Component - Date Picker

Overview

Date Picker is a powerful component that let the user select a specific date.

Element

Date Picker is created in HTML as below :
code

Attributes

Date Picker has 1 attribute :
lang (String): en | fr -> correspond to the lang of the date picker

CSS Parts

Date Picker doesn't have any part to customize at the moment.

Slots

Date Picker doesn't have any slots.

Events

Date Picker has 9 event to handle :
date-changed: fired when the selected date in the date picker has changed
confirm: fired when the user confirmed the selection of a date
cancel: fired when the user canceled the selection of a date
confirm-slider: fired when the user confirmed a month/year selection
cancel-slider: fired when the user canceled a month/year selection
confirm-keyboard: fired when the user selected a date using the keyboard input
cancel-keyboard: fired when the user canceled the selection of a date using the keyboard input
confirm-keyboard-scfl: fired when the user selected a date succesfully using the keyboard input
confirm-keyboard-wrong: fired when the user failed in selecting a date using the keyboard input
