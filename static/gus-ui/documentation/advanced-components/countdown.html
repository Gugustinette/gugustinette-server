Advanced Component - Countdown

Overview

Countdown is a component that displays a countdown.

Element

Countdown is created in HTML as below :
code

Attributes

Countdown has 8 attributes :
days (Int): number of days set to the counter
hours (Int): number of hours set to the counter
minutes (Int): number of minutes set to the counter
seconds (Int): number of seconds set to the counter
milliseconds (Int): number of milliseconds set to the counter
show-days (Boolean): tells if the days should be displayed on the counter
show-hours (Boolean): tells if the hours should be displayed on the counter
show-minutes (Boolean): tells if the minutes should be displayed on the counter

CSS Parts

Carousel has 1 parts to customize :
countdown: correspond to the countdown itself

Slots

Carousel doesn't have any slot.

Events

Carousel has 1 event to handle :
finished: fired when the countdown is over
