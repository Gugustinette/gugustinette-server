Advanced Component - Stepper

Overview

Stepper is a information component that is mean to display a defined number of steps.

Element

Stepper is created in HTML as below :
code

Attributes

Stepper has 4 attributes :
steps (JSON): JSON Array of objects (of type {title: "Step 1", content: "A"}, where title is the name of the step, and content is the text displaying in the circle representing the step) that represent each steps
nb-step (Int): correspond to the number of step to render
orientation (String): vertical | horizontal -> correspond to the orientation of the stepper
step-link (String): full | partial | none -> correspond to the way the steps are linked together

CSS Parts

Stepper has 1 part to customize :
stepper: correspond to the stepper itself
step: correspond to each circle representing a step
step-link: correspond to the link between each step
step-title: correspond to the area displaying the title of each step
step-title-text: correspond to the title of each step

Slots

Stepper doesn't have any slot.

Events

Stepper doesn't have any event to handle.
