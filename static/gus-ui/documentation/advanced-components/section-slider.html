Advanced Component - Section Slider

Overview

Section Slider is a powerful UI component that automatically creates a menu bar to navigate among sections. It is a strongly mobile inspired UI.

Element

Section Slider is created in HTML as below :
code

Attributes

Section Slider has 2 attributes :
sections (JSON): JSON Array of objects (of type {title: "Section 1"}) that represent the name of every section
inverted (Boolean): if true, the menu bar will be displayed at the bottom of the page.

CSS Parts

Section Slider has 5 parts to customize :
section-slider: correspond to the section slider itself
slider: correspond to the menu bar
slider-button-holder: correspond to the area holding the buttons
slider-line: correspond to the bar showing the actual section
section: correspond to the area where the sections are displayed

Slots

Section Slider can contain as many sections as you want. The sections should be some kind of divs which height and width are set to the window size, so the section slider can navigate through these properly.

Events

Section Slider doesn't have any event to handle.
