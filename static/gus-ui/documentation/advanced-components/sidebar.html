Advanced Component - Sidebar

Overview

Sidebar is a placement component that is to show elements on the side of the page when needed.

Element

Sidebar is created in HTML as below :
code

Attributes

Sidebar has 3 attributes :
visible (Boolean): tells if the sidebar is visible or not
direction (String): left | right -> correspond to the position of the sidebar on the screen
push-ui (Boolean): tells if the sidebar should push the whole UI or not

CSS Parts

Sidebar has 1 part to customize :
sidebar: correspond to the sidebar itself

Slots

Sidebar can contain as many elements as you want.

Events

Sidebar has 1 event to handle
statut-changed: fired when the state of the sidebar changed
