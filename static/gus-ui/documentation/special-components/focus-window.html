Special Component - Focus Window

Overview

Focus Window is a special UI component that is mean to display elements on top of the UI.

Element

A Focus Window should contain the elements that are supposed to be showing on top of the UI. Focus Window is created in HTML as below :
code

Attributes

Focus Window has 2 attributes :
visible (Boolean): tells if the focus window is visible or not
opacity (Float): from 0.0 to 1.0 -> correspond to the opacity of the focus window background

CSS Parts

Focus Window has 1 part to customize :
focus-window: correspond to the focus window itself

Slots

Focus Window can contain any element you want.

Events

Focus Window has 1 event to handle :
closed-window: fired when the user closes the focus window
