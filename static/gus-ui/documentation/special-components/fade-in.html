Special Component - Fade In

Overview

Fade In is a special effect that slowly display elements when they're reached by the user when scrolling.

Element

A Fade In should contain the element that is mean to have the fade in effect. Fade In is created in HTML as below :
code

Attributes

Fade In has 4 attributes :
offset (Int): correspond to the number of pixels that is not considered as visible at the bottom of the window. It is set to 100 by default, this means that if the element is in the last 100 pixels of the window height, the effect won't launch. It is mainly used to show elements when they hit near the middle of the window.
direction (String): top | bottom | left | right -> correspond to the direction of the effect
delay (Int): correspond to the delay before the effect launches (in ms)
speed (Int): correspond to the speed of the effect (in ms)

CSS Parts

Fade In has 1 part to customize :
fade-in: correspond to the area of the effect iself

Slots

Fade In can contain any element you want.

Events

Fade In has 1 event to handle :
fade-in: fired when the effect launches
