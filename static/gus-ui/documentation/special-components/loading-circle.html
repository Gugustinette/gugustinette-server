Special Component - Loading Circle

Overview

Loading Circle is a component that is mean to show the loading of a process.

Element

Loading Circle is created in HTML as below :
code

Attributes

Loading Circle has 2 attributes :
percentage (String): correspond to the percentage of completion of the circle
infinite (Boolean): tells if the loading circle should be of "infinite loading" type

CSS Parts

Loading Circle has 2 parts to customize :
loading-circle: correspond to the whole loading circle iself
value: correspond to the percentage value displaying inside the circle
circle: correspond to the circle itself

Slots

Loading Circle doesn't have any slot.

Events

Loading Circle doesn't have any event to handle.
