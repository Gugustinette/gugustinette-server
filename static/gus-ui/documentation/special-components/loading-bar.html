Special Component - Loading Bar

Overview

Loading Bar is a component that is mean to show the loading of a process.

Element

Loading Bar is created in HTML as below :
code

Attributes

Loading Bar has 1 attribute :
mode (String): normal | length-variation -> correspond to the style mode of loading bar

CSS Parts

Loading Bar has 2 parts to customize :
loading-bar-holder: correspond to the area holding the loading bar
loading-bar: correspond to the loading bar itself

Slots

Loading Bar doesn't have any slot.

Events

Loading Bar doesn't have any event to handle.
